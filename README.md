# PoE2 Rumor Counter — Установка и запуск на Windows 11 (nya~!)

## Требования
- **Python 3.9+** — скачайте с https://www.python.org/downloads/
  ⚠️ При установке ОБЯЗАТЕЛЬНО отметьте галочкой **"Add Python to PATH"**!
- **Windows 10/11** (OCR работает только на Windows)

---

## Быстрый запуск (рекомендуется)

Просто запустите файл **`run.bat`** двойным кликом. Он:
1. Проверит наличие Python в системе
2. Установит все зависимости автоматически
3. Запустит приложение

Всё, мяу~! (=^･ω・=)

---

## Ручная установка (для продвинутых)

```bash
# 1. Создайте виртуальную среду (рекомендуется)
python -m venv venv

# 2. Активируйте её
venv\Scripts\activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите приложение
python main.py
```

---

## Использование

1. **Нажмите "Select ROI & Start"** — появится прозрачное окно на весь экран
2. **Выделите мышкой область**, где появляются слухи в игре (например, всплывающие сообщения)
3. Нажмите Enter или отпустите кнопку мыши — сканирование начнётся!

**Горячая клавиша:** `F8` — остановка/запуск сканирования

---

## Возможные проблемы и решения 🐱

### ❌ "Python не найден"
- переустановите Python, обязательно отметив **"Add Python to PATH"** при установке
  
### ❌ "Ошибка при импорте 'PyQt6'"
- запустите `pip install -r requirements.txt` ещё раз
- убедитесь, что используете виртуальную среду (если создавали)

### ❌ "OCR не работает" / Приложение показывает "mock rumors" вместо реального текста
- проверьте, что установлены все зависимости из `requirements.txt`
- особенно пакет **`winsdk`** (он устанавливается только на Windows)
- запустите приложение с правами администратора
